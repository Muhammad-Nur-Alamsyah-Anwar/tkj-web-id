---
title: Tools Wajib
description: Daftar software dan tools yang wajib dimiliki anak TKJ.
---

Berikut adalah daftar tools yang wajib kamu install di laptopmu:

1. **VirtualBox** / **VMware** - Untuk virtualisasi.
2. **Cisco Packet Tracer** - Untuk simulasi jaringan Cisco.
3. **Winbox** - Untuk konfigurasi Mikrotik.
4. **Putty** - Untuk remote server via SSH.
5. **GNS3** / **EVE-NG** (Opsional) - Untuk simulasi jaringan advanced.
