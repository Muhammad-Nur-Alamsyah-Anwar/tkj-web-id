## ğŸš€ Cara Berkontribusi (Gampang Kok!)

Buat kamu yang baru pertama kali pakai Git/GitHub, ikuti langkah-langkah santai ini ya:

### 1. Fork Repository
Klik tombol **Fork** di pojok kanan atas halaman repository ini. Ini fungsinya buat nyalin project ini ke akun GitHub kamu sendiri.

### 2. Clone ke Laptop
Buka terminal/CMD/Git Bash di laptopmu, lalu ketik:
```bash
git clone https://github.com/USERNAME-KAMU/tkj-web-id.git
cd tkj-web-id
```
*(Ganti `USERNAME-KAMU` dengan username GitHub-mu ya)*

### 3. Buat Branch Baru
Jangan ngoding di branch `main` ya. Kita bikin cabang baru biar rapi.
```bash
git checkout -b tambah-materi-vlan
```
*(Namanya bebas, asal jelas)*

### 4. Mulai Ngoding/Nulis
Silakan edit file yang mau kamu ubah. Kalau mau nulis materi, file-nya ada di folder `src/content/docs/`.

### 5. Commit & Push
Sudah selesai? Simpan perubahanmu:
```bash
git add .
git commit -m "docs: menambahkan materi tentang vlan dasar"
git push origin tambah-materi-vlan
```

### 6. Bikin Pull Request (PR)
Balik ke halaman GitHub repository ini. Nanti bakal muncul tombol **"Compare & pull request"**. Klik aja, kasih judul yang jelas, dan kirim!
